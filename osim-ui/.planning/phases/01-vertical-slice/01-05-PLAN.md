Sidebar-Tree-Struktur (entsteht aus OtxJsonNode-Tree durch Gruppierung):
```
[Modell-Name]
├── Auslöser (N)
│   ├── Erzeugnis-1 [PAslEinzel]
│   └── ...
├── Durchlaufpläne (M)
│   ├── Plan-A [PDurchlaufplan]
│   │   ├── Knoten (K)
│   │   │   ├── Bearbeitung [PDpKnKonstant]
│   │   │   └── ...
│   │   └── Kanten (L)
│   └── Plan-B
│       └── ...
├── Ressourcen (R)
│   ├── Belegungsressourcen
│   ├── Mengenressourcen
│   └── Verknüpfungen
└── Schichten / Personal
    ├── Personal-Gruppen [AGruppe]
    └── Einsatzwünsche
```

Mapping JSON-Tree → Sidebar-Tree: ein Tree-Adapter funktion in `sidebar-tree.tsx`,
der `OtxJsonNode` → `react-arborist Node` (mit id, name, children, klass)
konvertiert und Gruppierungs-Header (synthetische Folder-Nodes) einfügt.

TYPE_MAP-Format:
```typescript
export interface PropertyMeta {
  octrl: 'variable' | 'bool' | 'enum' | 'link' | 'list' | 'method' | 'colorref' | 'logfont';
  type?: 'string' | 'int' | 'float' | 'bool';
  label: string;
  enumValues?: { value: string; label: string }[];
  linkTargetKlass?: string;
  listChildKlass?: string;
  readonly?: boolean;
}

export type ClassTypeMap = Record<string, PropertyMeta>;  // property → meta
export const TYPE_MAP: Record<Klass, ClassTypeMap>;
```
